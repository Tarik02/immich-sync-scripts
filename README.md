# immich-sync

## Usage

```bash
API_URL=https://immich.example.com \
API_KEY=api-key \
PERSON_ID=person-uuid \
ALBUM_ID=album-uuid \
./scripts/sync-to-album.js

API_URL=https://immich.example.com \
API_KEY=api-key \
REMOTE_URL=https://immich.other-person.com \
SHARE_KEY=share-key \
./scripts/sync-assets-from-shared.js
```
